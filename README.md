# Quantitative Trading Bot (Multi-Asset, ML-Enabled, Intraday)

This is a full-stack, intraday **quantitative trading bot** for equities, ETFs, crypto, forex, and options. It combines **machine learning**, **econometric modeling**, and **risk-managed execution logic**. The system is structured to mirror production-grade quant desks with modular design and academic-grade rigor.
